# Challenge Title : Stack and Queue

## Approach & Efficiency
### Stack big(O)
* push() : O(1)
* pop() : O(1)
* peek : O(1)
* is_empty : O(1)

### Queue big(O)
* inQueue() : O(1)
* deQueue() : O(1)
* peek() : O(1)
* is_empty() : O(1)

## Solution
==> python Stack/stack/stack.py
==> pytest Stack/tests/test_stack.py
==> python Queue/Queue/queue.py
==> pytest Queue/tests/test_queue.py
