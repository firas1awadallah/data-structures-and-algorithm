# Challenge Title : Stack and Queue

## Approach & Efficiency
### Stack big(O)
* push() : O(1)
* pop() : O(1)
* peek : O(1)
* is_empty : O(1)

### Queue big(O)
* inQueue() : O(1)
* deQueue() : O(1)
* peek() : O(1)
* is_empty() : O(1)

## Solution
1. ==> python Stack/stack/stack.py
2. ==> pytest Stack/tests/test_stack.py
3. ==> python Queue/Queue/queue.py
4. ==> pytest Queue/tests/test_queue.py
